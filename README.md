# Ejecutar en desarollo
1.  Clonar repositorio
2. Ejecutar

```bash
$ yarn install
```
3. Tener Nest CLI Instalado

```bash
$ npm i -g @nestjs/cli
```

4. Levantar la Base de datos
```bash
$ docker-compose up -d
```
## Stack utilizado 

* MongoDB
* Nestjs
